## <h1>Overview</h1>

<p className="dark:text-white text-black text-xl sm:text-4xl">
  Acme Auth is a full-stack authentication system built with React, Nextjs,
  Express, and MongoDB. It allows users to sign up, log in, and log out. The
  system also includes a password reset feature that sends an email to the user
  with a link to reset their password. The system is secure and uses JWT tokens
  for authentication.
</p>

## <h1> Features </h1>

- User authentication
- Sign up
- Log in
- Log out
- Password reset
- Secure JWT tokens
- User-friendly interface
- email verification

## <h1> Tech - Stack </h1>

- React
- Nextjs (App router)
- Express
- MongoDB
- JWT
